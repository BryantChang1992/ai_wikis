# 运营规范

> [!important] 负责人
> **COO** 负责修改。跨领域变更需 CEO 审批。

## 适用范围

运营规范约束运营团队（COO → 专家·数据分析/流程优化 → 执行层），包括但不限于：

- 跨团队协作流程
- 数据分析标准与指标体系
- 流程优化方法论
- 效率度量标准
- 知识管理流程（文档沉淀、Wiki 维护）
- 会议与沟通规范
- 日报/周报/月报模板
- 项目管理框架（Scrum/Kanban 等）

## 关联 Agent

- **COO** — 最终负责人
- 专家·数据分析 — 数据规范
- 专家·流程优化 — 流程规范
