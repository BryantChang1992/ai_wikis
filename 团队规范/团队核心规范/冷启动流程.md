# 冷启动流程

Agent 首次启动或 compaction 恢复后执行：

1. `git pull` ai_wikis（拉最新规范）
2. 加载自身 AGENTS.md + SOUL.md（已注入）
3. 加载领域规范：`read work/ai_wikis/团队规范/` 下对应领域
4. 加载短期记忆：memory_search 查询 MEMORY.md
5. Supervisor 对账：检查是否有关键未完成任务

注：AGENTS.md/SOUL.md 每轮自动注入 system prompt，无需手动 read。其他规范文件按需 read。
