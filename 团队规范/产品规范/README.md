# 产品规范

> [!important] 负责人
> **CPO** 负责修改。跨领域变更需 CEO 审批。

## 适用范围

产品规范约束产品团队（CPO → 专家·用户研究/产品设计 → 执行层），包括但不限于：

- 产品 Roadmap 规划流程
- 需求文档编写规范
- 用户研究方法论与执行标准
- 产品设计原则与 Design System
- 竞品分析框架
- 产品发布流程
- 用户反馈收集与处理机制
- 产品指标体系（OKR / KPI）

## 关联 Agent

- **CPO** — 最终负责人
- 专家·用户研究 — 调研规范
- 专家·产品设计 — 设计规范
