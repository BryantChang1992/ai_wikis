# 通信协议规范

## 层级通信规则

| 场景 | 方式 | Context |
|------|------|---------|
| 上级 → 下级委派 | `sessions_spawn` | `isolated` |
| 同级协作 | `sessions_send` | 不创建新 session |
| 需要上游上下文 | `sessions_spawn` | `fork`（谨慎） |
| Supervisor 干预 | `sessions_send` | 对账、异常回收 |

## 模型分配

- VP 层 & 专家层: `deepseek/deepseek-v4-pro`
- Worker 层: `deepseek/deepseek-v4-flash`

## sessions_spawn 知识传递

子 Agent 不知道父 Agent 上下文。派发任务时需在 task 中嵌入知识引用（告知查阅什么规范文件）。
