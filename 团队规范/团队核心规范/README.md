# 团队核心规范

> [!important] 负责人
> **CEO (Mike)** + **CTO** 共同负责修改。
> 其余 VP 可建议，需 CEO/CTO 审批后生效。

## 适用范围

团队核心规范约束所有 Agent，是对 **CHANG_AI_TEAM 整体运作方式** 的定义，包括但不限于：

- Agent 组织架构与层级关系
- 角色定义与汇报关系
- 通信协议（spawn/send/fork 使用规则）
- 任务状态机与生命周期
- 冷启动流程
- 仓库分工与 Git 操作规范
- 模型分配规则

## 与其他规范领域的关系

```
团队核心规范 ← 最高层级，所有 Agent 必须遵守
├── 技术规范  ← CTO 负责
├── 产品规范  ← CPO 负责
├── 财务规范  ← CFO 负责
└── 运营规范  ← COO 负责
```

各领域规范不得与团队核心规范冲突。如有冲突，以团队核心规范为准。
