# 技术规范

> [!important] 负责人
> **CTO** 负责修改。跨领域变更需 CEO 审批。

## 适用范围

技术规范约束技术团队（CTO → 专家·Infra/Perf/SRE → 执行层 RD/SRE/QA），包括但不限于：

- 技术选型标准与决策流程
- 代码规范与 Code Review 流程
- 架构设计原则
- 基础设施标准（部署、监控、CI/CD）
- 技术债务管理
- 事故响应流程（on-call、升级、复盘）
- API 设计规范
- 数据安全与隐私技术标准

## 关联 Agent

- **CTO** — 最终负责人
- 专家·Infra — 基础设施规范
- 专家·Perf — 性能规范
- 专家·SRE — 可靠性规范
