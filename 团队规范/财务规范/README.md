# 财务规范

> [!important] 负责人
> **CFO** 负责修改。跨领域变更需 CEO 审批。

## 适用范围

财务规范约束财务团队（CFO → 专家·预算/审计/税务 → 执行层），包括但不限于：

- 预算编制与审批流程
- Token 成本核算与归因标准
- 财务数据报告规范
- 审计流程与合规检查
- 采购与供应商管理
- 成本优化策略
- 财务 Dashboard 指标定义

## 关联 Agent

- **CFO** — 最终负责人
- 专家·预算 — 预算规范
- 专家·审计 — 审计规范
- 专家·税务 — 税务规范
