# agent基建需要解决的核心问题
从agent视角
1. 记忆管理
	1. 长短期的记忆，记忆和agent间的关系
	2. 高效检索，召回
2. 工作流自动化
	1. agent间的协作规范--各层级agent的权限，边界
	2. 任务状态的管理和审计
3. 成本追踪--token

类比公司，大致是这么几个维度
agent维度--人的维度：
	agent的层级，各种规章制度，汇报关系，任务派发关系，沟通机制等
任务维度：
	任务的追踪，拆解，状态转移
成本维度：
	整体的成本构成，如何拆分到各个团队，各个项目
		核心计量单位：token
知识管理：
	wiki管理，检索
	agent记忆管理---这个对应公司，或者人应该是什么呢？
